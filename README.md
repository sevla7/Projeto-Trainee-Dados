# House Prices - Predição de Preços de Imóveis

Projeto desenvolvido durante o processo Trainee da **CATI Jr.**, com o objetivo de aplicar técnicas de Ciência de Dados e Machine Learning para prever o preço de venda de imóveis utilizando o conjunto de dados **House Prices: Advanced Regression Techniques**.

O projeto contempla todas as etapas de um fluxo de trabalho de Data Science, desde a análise exploratória dos dados até a construção, avaliação e comparação de modelos preditivos, seguindo boas práticas de desenvolvimento e documentação.

## Objetivos

- Compreender o problema de negócio;
- Realizar Análise Exploratória de Dados (EDA);
- Tratar valores ausentes e inconsistências;
- Desenvolver engenharia de variáveis;
- Preparar dados numéricos e categóricos;
- Construir e comparar modelos de regressão;
- Avaliar o desempenho utilizando métricas apropriadas;
- Gerar previsões para novos dados.

## Tecnologias Utilizadas

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- Jupyter Notebook

## Modelos Utilizados

- Regressão Linear
- Ridge Regression
- Random Forest Regressor
- Gradient Boosting Regressor

## Técnicas Aplicadas

- Análise Exploratória de Dados (EDA)
- Estatística Descritiva
- Tratamento de Valores Ausentes
- Engenharia de Variáveis
- One-Hot Encoding
- Padronização de Dados
- Pipeline de Pré-processamento
- Validação Cruzada
- Comparação de Modelos
- Avaliação com MAE, RMSE, RMSLE e R²

## Aprendizados

Este projeto permitiu consolidar conhecimentos em análise de dados, engenharia de atributos, modelagem preditiva e avaliação de modelos de regressão, além de reforçar a importância do pré-processamento e da interpretação dos resultados para a construção de soluções baseadas em dados.
